**Бэкенд:** [https://shoe-store-18si.onrender.com](https://shoe-store-18si.onrender.com)

## Функции
- Каталог товаров с категориями
- Поиск по названию и цвету
- Хиты продаж
- Страница товара с выбором размера и количества
- Корзина (сохранение в localStorage)
- Оформление заказа
- Загрузка товаров по 6 (кнопка «Загрузить ещё»)
- Переключение вида корзины (список / плитка)
- Изменение количества товаров в корзине (+/−)
- Валидация телефона и адреса при заказе
- Адаптивный дизайн
- Обработка ошибок с повторной загрузкой

## Стек
- **React 19** + **TypeScript**
- **Vite 8**
- **Bootstrap 5**
- **React Router 7**
- **Context API** + **useReducer**
- **Render** (бэкенд)
